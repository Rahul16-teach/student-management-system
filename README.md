# student-management-system
## Project Overview
Student Management system is a Django-based backend project developed to manage student records efficiently and securely.
The project is implemented in two parts: a Django Web Application and its REST API version using Django Rest Framework....
-------

## Features
### Django Web Application
- Admin authentication using Django auth system
- Add, update, delete student records
- Secure access for authorized users
- Database handled using Django ORM
  
  ### REST API (DRF)
  - RESTful CRUD APIs for student data
  - Data validation using serializers
  - JWT-based authentication
  - API testing using Postman

  --------

## Tech stack
- Python
- Django
- Django REST Framework
- SQLite
- JWT Authentication
- HTML, CSS(for web version)

  --------

## Project Structure
- Django Models for database design
- views for business logic
- URLs for routing
- Serializers for API validation
- JWT for secure authentication

  -----------

## How to Run the Project
1. clone the repository
2. create virtual environment
3. Install dependencies
4. Run migrations
5. Start development server
